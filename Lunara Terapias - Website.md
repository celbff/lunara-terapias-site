# Lunara Terapias - Website

## Descrição
Website profissional para a Lunara Terapias, um centro de terapias integrativas e práticas holísticas. O site apresenta uma abordagem zen e minimalista, com design responsivo e funcionalidades interativas.

## Características Principais

### Design
- **Paleta de cores**: Baseada no degradê suave do bege ao verde, transmitindo paz e equilíbrio
- **Tipografia**: Poppins (Google Fonts) para uma aparência moderna e legível
- **Layout**: Design responsivo que se adapta a dispositivos móveis e desktop
- **Estilo**: Zen, minimalista e acolhedor

### Seções do Site

#### 1. Hero Section
- Logo da Lunara Terapias
- Título principal e tagline
- Descrição dos serviços
- Botões de call-to-action (Conheça as Terapias e Lunara Tarô)

**Radiestesia e Radiônica:**
- Radiestesia

**Oráculos e Cartomancia:**
- Tarô

**Astrologia e Psicologia:**

#### 3. Sobre
- Descrição da filosofia da Lunara Terapias

### Funcionalidades

#### Interatividade
- Menu de navegação responsivo com hamburger menu para mobile
- Navegação suave entre seções
- Formulário de contato com validação
- Sistema de notificações
- Efeitos de hover e animações
- Botão "scroll to top"

#### Responsividade
- Layout adaptável para diferentes tamanhos de tela
- Menu hamburger para dispositivos móveis
- Imagens e textos otimizados para mobile

#### Performance
- Carregamento otimizado de imagens

## Estrutura de Arquivos

```
lunara_terapias/
├── index.html          # Página principal
├── style.css           # Estilos CSS
├── script.js           # JavaScript interativo
├── logo_lunara.jpg     # Logo principal com texto
├── logo_contorno.png   # Logo símbolo (contorno)
```

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Estilos modernos com flexbox e grid
- **JavaScript**: Interatividade e funcionalidades
- **Font Awesome**: Ícones
- **Google Fonts**: Tipografia (Poppins)

## Como Usar

1. Abra o arquivo `index.html` em um navegador web
2. O site é totalmente funcional localmente
3. Para publicação, faça upload dos arquivos para um servidor web

## Personalização

### Cores
As cores podem ser facilmente alteradas no arquivo CSS, baseadas na paleta:
- Verde principal: #4a7c59
- Verde secundário: #6ba777
- Bege claro: #f5f1e8
- Verde água: #e8f5f0

### Conteúdo
Todo o conteúdo pode ser editado diretamente no arquivo HTML, incluindo:
- Textos das terapias
- Depoimentos
- Informações de contato
- Links de redes sociais

## Compatibilidade

- **Navegadores**: Chrome, Firefox, Safari, Edge (versões modernas)
- **Dispositivos**: Desktop, tablet, smartphone
- **Resolução**: Otimizado para 320px a 1920px+
